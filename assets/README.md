# Cat Video Asset

The extension looks for the cat break video at:

- `assets/cat-side.mp4`
- `assets/cat_1.mp4`
- `assets/cat_3.webm`
- `assets/cat.webm`
- `assets/cat.mp4`

Use a short, looping, clearly licensed cat video. WebM is preferred; MP4 is supported as a fallback. If the file is missing, the extension still works and displays an animated CSS fallback, but the intended production experience is to bundle a realistic video here.

## Bundled asset

- Active file: `assets/cat_8.mp4`
- Purpose: composited cat overlay. The content script processes this source through a canvas keying/matting pipeline so the original background is removed at runtime before the cat is placed into the page space.
